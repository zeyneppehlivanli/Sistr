# Sistr - Kampüs İçi Güvenlik ve Eşleşme Ağı

## Problem
Üniversiteli kadın öğrencilerin kampüste yalnız hissiyatı, özellikle geç saatlerde kütüphaneden dönerken yaşanan güvenlik endişeleri ve güvenli sosyalleşme bariyerleri.

## Çözüm
Sistr, sıradan bir ilan panosundan ziyade, kendi "karar mekanizması" olan dijital bir kız kardeşlik ağıdır. Kullanıcılar filtreli etkinliklere katılabilir. En önemli farkı olan "Guardian Mode" sayesinde, acil gece yürüyüşü ilanları tüm filtreleri ezer (override) ve buluşmalarda arka planda "Güvende misin?" (Live Safety Check) algoritması çalıştırarak gerektiğinde SOS gönderir.
